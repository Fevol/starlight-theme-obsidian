This theme is mainly designed to recreate the look and feel of the [Obsidian Publish](https://obsidian.md/publish) websites.
If you are looking for additional functionality, check out a selected list of tools and templates that can be used with this theme.
Make sure to open an issue or PR if you wish for another tool to be added to this list.

### Plugins
The list below contains additional packages that can provide useful functionality to your Starlight site.
For a complete list of all registered plugins for Starlight, check out the [Starlight Plugin Registry](https://starlight.astro.build/resources/plugins/).
